✨ Бот теперь видит картинки в .docx:
• Вставь картинку прямо в статью ПЕРЕД маркером Рис. — бот возьмёт её как референс для этого слота
• Можно писать «Рис. Используй контент из картинки выше» — работает

📊 Парсит markdown-таблицы:
• Под маркером Рис. таблицу в .docx → каждая строка становится буллетом → инфографика по содержимому таблицы

💡 В смете теперь видно:
• 📎 у каждого слота с inline-рефом из .docx
• ⚠️ если в тексте ссылка «из картинки выше», но картинку не нашли

🔢 Каждая отдаваемая картинка подписана «X из N»:
• На первой подсказка «нажми ✅ ОК для следующей»

🐛 Починили:
• Стиль ссылается на компоненты рефов (палитра, материалы, фон), не копирует субъект буквально
• Заголовок и номерные блоки в картинке только при 4+ буллетах (раньше уже при 2)
• Можно явно указать `Рис.[grid]` или `Рис.[scene]` в статье
