# Бот @tychyna_images_bot — установка (двойной клик, ~3 минуты)

Один раз поставил — дальше бот сам включается при каждом входе в мак.
Пользоваться можно с телефона: просто пиши боту в Телеграм.

## Установка — 2 шага

1. Распакуй папку `article-img-bot` куда удобно (напр. в Документы).
2. **Двойной клик по `install.command`**.
   - Если macOS ругается «неизвестный разработчик» — это нормально для
     своих скриптов: **правый клик по `install.command` → Открыть →
     Открыть**. Нужно сделать так **только первый раз**.
3. Дождись `✅ Готово` в открывшемся окне. Всё.

После этого ничего делать не надо — бот работает и сам стартует
при каждом включении мака.

## Пользоваться — всё кнопками, команды не нужны

Пиши боту **@tychyna_images_bot** в Телеграме:

- **Картинки к статье** — кинь файл (`.docx`/`.md`/`.txt`) или ссылку.
  Бот покажет смету → жми «🚀 Запустить» → через пару минут пришлёт
  ZIP с картинками.
- **Свои образцы стиля (референсы)** — просто пришли фото-картинку.
  Бот спросит кнопками «🧮 Инфографика / 🎬 Сюжет» — тапни нужное,
  фото сохранится в библиотеку.
- **Перевод картинок** — после готовой пачки бот покажет кнопку
  «🌍 Перевести». Тапнул → пришли статью с переведённым текстом в
  том же формате → бот перерисует те же картинки с новым текстом.

## Кнопки на десктопе (двойной клик)

- `status.command` — проверить, работает ли бот
- `stop.command` — остановить бота и убрать автозапуск

## Важно (честно)

- Бот живёт, пока **мак включён и не спит**. Закрытая крышка усыпляет —
  для постоянной работы держи мак открытым / на зарядке. Это
  ограничение «локального» бота, никак не обойти.
- Файл `.env` с ключами уже лежит в папке. Не удаляй, без него бот
  не запустится.
- Удалить или почистить рефы — открой папку `refs/` в Finder, удали
  ненужные файлы вручную.
